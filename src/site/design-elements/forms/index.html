Text Inputs

Below is the style for a left-aligned form, with labels and inputs on the same line. You'll want to tweak the width and proportion of your form, based on the label length and required text input. Choose an appropriate column width for the form class, and then adjust the col-xs-* classes such that the total number in each form-group is 12.

If you have limited horizontal space, or don't want to deal with left-aligned lables, there is a second form style. The text inputs will be 100% width of the form, so control their widths with the col-xs-* in the form tag.

If you need to input a bunch of text, try a textarea!
